Project


CamProctor Web App for Electronic Written Exams Proctoring System


CamProctor is an advanced web application designed to ensure the integrity of electronic written exams by using Temporal Convolutional Networks (TCNs) for robust proctoring. It offers real-time monitoring with live video analysis, automatically detecting and alerting on suspicious activities. The system features a user-friendly interface and ensures high accuracy in anomaly detection. CamProctor is built using Python, MySql, Bootstarp and Flask, and can be seamlessly integrated with existing Learning Management Systems (LMS), making it an essential tool for maintaining academic honesty in the digital examination process.
